# 1 匹配字符串

正则表达式的符号主要分为`元字符`和`表示重复的字符`，分别如下表所示

==元字符==

| 字符 |                     备注                      |
| :--: | :-------------------------------------------: |
|  .   | 除换行符以外所有字符，若要匹配“.”需要使用转义 |
|  \w  |              字母、数字、下画线               |
|  \s  |                空白符（空格）                 |
|  \d  |                     数字                      |
|  ^   |                 表达式的开始                  |
|  $   |               匹配表达式的结束                |



==表示重复的字符==

|  字符  |   备注   |
| :----: | :------: |
|   *    |  ≥ 0 个  |
|   ？   | 0 ~ 1 个 |
|   +    |  ≥1 个   |
|  {n}   |   n 个   |
| {n, M} | n ~ M 个 |
| {n, }  |  ≥ n 个  |

<br>

string str = "Hello World";

if(regex.IsMatch(str)){

}
