    FPGA入门笔记(一)：Intel FPGA开发流程

    Krins 12/24/2021 fpgaalteraverilog

    # 1.创建文件夹

    doc：设计相关文档存放目录

    img：设计相关图片存放目录

    prj：工程文件存放目录

    rtl： verilog 可综合代码存放目录

    testbench：测试文件存放目录

    # 2.创建工程

    1. New Project Wizard -> Next

    1. 选择项目路径，填写工程名称，点击Next

    1. 选择Empty Project，点击Next

    1. 点击Next

    1. 选择自己的芯片型号，点击Next

    1. 设置仿真工具为ModelSim - Alter，仿真语言为Verilog HDL，点击Finish完成工程创建

    # 3.添加设计文件

    1. 点击 New 创建新文件

    1. 选择文件类型，点击 OK

    1. 输入相关代码，保存在 rtl 文件夹下

    1. 点击 Start Analysis & Synthesis 进行仿真和综合

    1. 点击 Tools -> Netlist Viewers -> RTL Viewer，查看硬件逻辑电路

    # 4.功能仿真

    1. 新建 .v 文件，并保存在工程对应的 testbench 文件夹下

    1. 点击标题栏 Assignments -> Settings -> Simulation，检查信息是否与设置的一致

    1. 选择 Compile test bench，点击右侧 Test Benches...

    1. 点击 New

    1. 选择已编辑好的 testbench 文件，点击 Add

    1. 填写 Test bench name，需要与之前添加的 testbench 文件同名，点击 OK

    1. 点击 OK

    1. 点击 OK

    1. 点击 Tools -> RTL Simulation Tool -> RTL Simulation，进行功能仿真（理想情况）

    1. 查看仿真结果

    1. 若仿真失败，点击 View -> Transcript 查看错误原因

    # 5.综合与布局布线

    1. 点击主界面 Start Compilation

    1. 点击标题栏 Tools -> Run Simulation Tool -> Gate Leval Simulation，进行门电路仿真（非理想情况）

    1. 选择时序模型，点击 Run

    1. 查看仿真结果，发现与之前的功能型仿真结果不同

    # 6.分配引脚

    1. 点击菜单栏 Pin Planner

    1. 根据实际情况配置引脚

    1. 关闭引脚配置窗口，点击 Start Compilation

    # 7.下载

    # 非固化下载
    1. 点击菜单栏 Programmer

    1. 点击 Hardware Setup...

    1. 选择 USB - Blaster

    1. 点击左侧 Auto Detect，选择 EP4CE10

    1. 点击 Add File...

    1. 选择工程目录下 output_files 文件夹下的 .sof 文件

    1. 点击 Start，等待下载结束

    # 固化下载
    1. 点击标题栏 File -> Convert Programming File

    1. 设置如下选项

    1. 点击 Flash Loader，并在右侧点击 Add Device...

    1. 选择相应设备，点击 OK

    1. 点击 SOF Data，并在右侧点击 Add File...

    1. 选择之前生成的 .sof 文件，点击 Generate

    1. 重复上述 “ 非固化下载 ” 步骤，将下载文件由 .sof 替换为 .jic 即可

    1. 重新上电即可查看效果

    # 8.基于vs code开发

    1. 在testbench文件中添加代码

      initial begin
    //指定vcd文件的名字为wave.vcd
    //仿真信息将记录到build文件夹下的wave.vcd文件中
	$dumpfile("./build/wave.vcd");  
    
    //xxx_tb: testbench模块名称
    $dumpvars(0, xxx_tb);  
	
    //...
end

    2. 创建文件夹build

      mkdir build
      1

    3. 编译模块

      iverilog -o ./build/output_file_name.out ./testbench/testbench_module_name.v ./rtl/src_module_name.v
      1

    4. 根据.out文件生成vvp波形

      vvp -n ./build/output_file_name.out
      1

    5. 使用gtkwave工具查看仿真波形

      gtkwave ./build/wave.vcd
      1

    6. 或直接使用python脚本

      import os

# 编译文件
print("[1/3] Compile the module...")
src_file = input("Source:")
tst_file = input("Testbench:")
out_file = input("Out:")
os.system("iverilog -o ./build/" + out_file + ".out " + "./testbench/" + tst_file + ".v " +  "./rtl/" + src_file + ".v")

# 生成vvd波形
print("[2/3] Generate vvd wave...")
os.system("vvp -n ./build/" + out_file + ".out")

# 查看波形
print("[3/3] Show wave...")
wave = input("WaveName:")
os.system("gtkwave ./build/" + wave + ".vcd")
